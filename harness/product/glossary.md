# Product Glossary

Use this glossary when writing or reviewing user-visible Dear Hoomin copy.

- Dear Hoomin: product name.
- hoomin: product-facing word for a person in the family.
- furbaby: product-facing word for a pet in family/profile settings.
- musing: product-facing word for generated pet-voice entries.
- daily musing: scheduled daily generated pet ritual.
- journal musing: pet-voice entry created from a hoomin's journal note and photos.
- thought: internal data model term. Use it in code, schemas, and technical docs when referring to the durable `thoughts` model or when the implementation distinction matters.
- avatar: product-facing word for the selected identity image for a furbaby, hoomin, or future avatar-able subject.
- referenced avatar: a hoomin avatar with a family-scoped reference name such as poppa, mooma, or a custom name, so generation can include that hoomin when musing or journal text names them.
