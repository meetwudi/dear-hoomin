# Referenced Hoomin Avatars

## Alignment

- A hoomin avatar can have a family-scoped reference name.
- Use "referenced avatar" for the concept of an avatar that generation can identify from text.
- Suggested reference names can be playful family words such as poppa or mooma, but a hoomin can customize the name.
- When a musing or journal note names a reference name, image generation should prefer that hoomin's avatar/photo as the hoomin identity anchor.
- If no reference name is mentioned, generation can still use the requesting hoomin's avatar/photo when available.
- Reference names are generation hints, not auth roles or permission roles.
- Avatar identity preservation means the subject remains recognizable; clothing can change to fit the avatar option or generated scene.
