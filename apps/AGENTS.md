# AGENTS.md

## Scope

This file applies to `apps/`.

## Instructions

- Use this folder for deployable application surfaces.
- Keep each app in its own folder with local `AGENTS.md` guidance.
- Keep durable product and architecture decisions in `harness/`.
- Check and update `harness/platform-dependencies.md` when app code adds or changes provider-specific integrations, deployment behavior, scheduled jobs, storage, auth providers, or env vars.
