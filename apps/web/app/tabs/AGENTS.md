# AGENTS.md

## Scope

This file applies to `apps/web/app/tabs/`.

## Instructions

- This folder records the product contract for primary app tabs.
- The only primary app tabs are `Musings` and `Family`.
- Each primary tab must have a folder here with its own `AGENTS.md`.
- Do not add, rename, or remove primary tabs unless the developer explicitly confirms the tab model change.
- Keep tab-owned settings inside the owning tab instead of adding a duplicate settings tab or duplicate settings route.
- If a change touches tab navigation, check `harness/required-features.md` before editing and update it in the same change when the developer confirms a product change.
