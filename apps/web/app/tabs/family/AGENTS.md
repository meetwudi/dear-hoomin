# AGENTS.md

## Scope

This file applies to the `Family` primary app tab.

## Instructions

- `Family` owns family members, invites, furbaby profile setup, avatar selection, real-world photo replacement, furbaby notes, timezone preferences, and notification preferences.
- Keep browser notification enablement and daily-musing-ready notification preferences in this tab.
- Do not create a separate Furbaby or Settings primary tab for these controls unless the developer explicitly confirms the tab model change.
